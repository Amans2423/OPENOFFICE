Office Tools – Concepts + Olympiad Level Quiz

1. Licensed, Proprietary and Open-Source Software

  • Licensed Software: Paid software; you must buy a license to use it (e.g., MS Office).
  • Proprietary Software: Owned by a company; source code not available.
  • Open-Source Software: Free to use; source code available (e.g., OpenOffice, LibreOffice).
  • Freeware: Free to use but cannot modify (e.g., Adobe Reader).
  • Shareware: Trial version, limited time use.
  • Examples in real life: Linux (open-source), Windows (proprietary), VLC player (open-source).

2. Writer (Word Processor)

  • Create text documents (letters, reports, assignments).
  • Saving in multiple formats (.odt, .docx, .pdf).
  • Formatting: Bold, Italic, Underline, Font size, Alignment.
  • Page formatting: Margins, Header, Footer, Page numbering.
  • Printing options: Print all pages, selected pages, preview.
  • Mail Merge: Auto-filling names & data into letters.
  • Toolbars: Standard, Formatting, Table toolbar.
  • Shortcut keys: Ctrl+S (Save), Ctrl+P (Print), Ctrl+B (Bold), Ctrl+Z (Undo).

3. CALC (Spreadsheet)

  • Cell: Single box in spreadsheet.
  • Cell Address: Column letter + Row number (e.g., A1, C5).
  • Formula: Begins with '=' sign.
  • Common formulas: =SUM(A1:A5), =AVERAGE(B1:B4).
  • Formatting: Borders, background color, number formats.
  • Toolbars: Formula bar, Standard toolbar, Formatting toolbar.
  • Charts/Graphs can be created from data.

4. Impress (Presentation Software)

  • Slide: One page in a presentation.
  • Transitions: Animation effects applied between slides.
  • Views: Normal view, Slide sorter, Slide show view.
  • Shortcuts: F5 (Start slideshow), Ctrl+M (New slide).
  • Saving, Opening, Closing: Save → .odp or .pptx.
  • Toolbars: Slides pane, Tasks pane, Formatting, Drawing toolbar.
  • Printing: Print slides, handouts, notes.
